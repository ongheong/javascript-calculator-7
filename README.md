### 🔧 구현할 기능
1. 테스트 케이스를 기본 구분자로 파싱, 계산하여 출력
2. 사용자 입력을 받아 처리 루틴에 따라 출력
3. 커스텀 구분자를 지정하여 처리 루틴에 따라 출력
4. 입력 예외 처리
